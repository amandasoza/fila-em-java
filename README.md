Fila em Java

Este repositório contém uma implementação simples de uma fila em Java utilizando uma estrutura de lista encadeada.
Descrição

A fila segue o princípio FIFO (First In, First Out), ou seja, o primeiro elemento a entrar é o primeiro a sair. Ela usa uma lista encadeada para organizar os elementos, onde cada elemento é um nó com dados e um ponteiro para o próximo nó.
Funções principais:

    create: Cria a fila vazia.

    enqueue: Adiciona um novo valor no final da fila.

    dequeue: Remove o primeiro valor da fila.

    isempty: Verifica se a fila está vazia.

    destroy: Limpa a fila, removendo todos os elementos.

    display: Exibe todos os elementos da fila.
